# mac 开发环境配置

- 安装 `brew` [参考网址](https://brew.sh/) 

```text
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

- 安装 `java`

```text
brew cask install java
```

- 安装 `tomcat`

```text
brew search tomcat

brew install tomcat
```

- rsa public key

```text
ssh-keygen -t rsa

cat ~/.ssh/id_rsa.pub
```
